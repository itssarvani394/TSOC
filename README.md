# TSOC Workshop

## Technologies Used
- Frontend: HTML, CSS, JavaScript
- Toolkit: bun
  
## Installation
Copy the Github URL and Clone it in VS-Code using:
```bash
  git clone https://github.com/itssarvani394/TSOC.git
```

Install the necessary dependencies using:
```bash
bun i neucron-sdk

bun i @bsv/sdk
```

Run the files using:
```bash
bun run workshop.js

bun run bsvsdk.js
```
